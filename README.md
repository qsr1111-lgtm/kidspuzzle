# Котики Пазлы

Готовое PWA-приложение для детей 3–5 лет.

## Запуск на Mac

1. Распакуй ZIP на Рабочий стол.
2. Открой Терминал.
3. Выполни:

```bash
cd ~/Desktop/kids-puzzles
npm install
npm run dev
```

4. Открой в браузере:

```txt
http://localhost:5173
```

## Сборка

```bash
npm run build
```

## Деплой на Vercel

- Framework: Vite
- Build command: npm run build
- Output directory: dist
